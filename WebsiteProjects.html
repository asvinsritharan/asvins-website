Project Highlights

Facial Recognition

Description

This is a script which learns faces based on facial characteristics. The user provides the script with a location which contains images of people, then labels them through prompts in the script. A picture can then by provided to the script of a person previously labelled by the user. The script will match facial characteristics of the face in the picture with the model created by the script to determine the matching label of the person in the user provided picture.

Language

Python

Classification Method

Principal Component Analysis (PCA)

Number Identifier

Description

This script uses the MNIST written number dataset to train the KNN model. Upon completion of the creation of the model, the user can provide a number to the script when prompted, then the model will provide it's prediction to the number that was written on the image.

Language

Python

Classification Method

K Nearest Neighbours (KNN)
